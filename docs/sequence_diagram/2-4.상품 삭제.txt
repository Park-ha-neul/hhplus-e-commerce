title 상품 삭제

관리자->상품: 상품 조회 요청 
activate 상품
상품->상품: 관리자인지 검증
alt 관리자가 아닌 경우
  상품-->관리자: 오류 응답 조회
else 관리자인 경우
  상품->상품: 상품 검증
  alt 상품 검증 실패
    상품-->관리자: 오류 응답 조회
  else 상품 검증 성공
    상품->DB: 상품 삭제
    activate DB
    DB-->상품: 상품 삭제 응답
    deactivate DB
    alt 상품 삭제 완료
      상품-->관리자: 상품 삭제 성공 응답
    else 상품 삭제 실패
      상품-->관리자: 상품 삭제 실패 응답
    deactivate 상품
    end
  end
end
