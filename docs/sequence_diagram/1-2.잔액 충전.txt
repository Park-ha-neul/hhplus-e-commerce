title 잔액 충전

사용자->포인트:잔액 충전 요청
activate 포인트
포인트->포인트:충전 포인트 검수
alt 1회 충전량 초과
    포인트 --> 사용자 : 오류 응답 반환 (충전량 초과)
else 최대 한도 초과
    포인트 --> 사용자 : 오류 응답 반환 (최대 한도 초과)
else 음수 충전 요청
    포인트 --> 사용자 : 오류 응답 반환 (음수 값)
else 충전 가능
  포인트->DB: 잔액 업데이트
activate DB
  DB-->포인트: 업데이트 된 잔액
deactivate DB
포인트->포인트: 충전 전/충전 후 금액 검증
  alt 업데이트 성공 
    포인트-->사용자: 충전 완료 응답
  else 업데이트 실패
    포인트-->사용자: 오류 응답 반환 
  deactivate 포인트
  end
end
