title 상품 수정

관리자->상품: 상품 수정 요청 
activate 상품
상품->상품: 관리자인지 검증 
alt 관리자가 아닌 경우 
  상품-->관리자: 오류 응답 조회 
else 관리자인 경우 
  상품->상품:상품 검증
  alt 등록된 상품이 아닌 경우 
    상품-->관리자: 오류 응답 조회
  else 등록된 상품인 경우 
    상품->DB: 상품 수정
    activate DB
    DB-->상품: 수정된 상품 정보 응답
    deactivate DB
    alt 상품 수정이 완료된 경우
      상품-->관리자: 수정된 상품 정보 응답 
    else 상품 수정 실패한 경우 
      상품-->관리자: 오류 응답 조회
      deactivate 상품
    end
  end
end
