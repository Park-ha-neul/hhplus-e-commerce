title 상품 등록

관리자->상품: 상품 등록 요청 

activate 상품
상품->상품: 관리자인지 검증 

alt 관리자가 아닌 경우
  상품-->관리자: 오류 응답 조회
else 관리자인 경우
  상품->DB: 상품 등록
  
  activate DB
  DB-->상품:등록된 상품 정보
  deactivate DB
  alt 상품 등록이 완료된 경우 
    상품-->관리자: 등록된 상품 정보 응답 
  else 상품 등록 실패한 경우
    상품-->관리자: 오류 응답 조회 
  end
  
  deactivate 상품
  
end
